Información del Proyecto

Gas IA Bolivia es una plataforma web desarrollada para mejorar el control de garrafas de GLP, permitiendo organizar registros, visualizar información, monitorear estados y apoyar la toma de decisiones mediante reportes y alertas del sistema.

Módulos del Sistema

📊 Dashboard

Visualización general de indicadores.

🛢️ Gestión de Garrafas

Registro y administración de garrafas.

📡 Monitoreo

Seguimiento del estado de las garrafas.

⚠️ Alertas IA

Alertas relacionadas con eventos importantes.

📄 Reportes

Reportes para la gestión de información.

👤 Usuarios

Acceso mediante autenticación.
